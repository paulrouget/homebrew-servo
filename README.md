homebrew-servo
===============

Homebrew formula for [Servo](http://servo.org).

## Usage:

    $ brew tap paulrouget/servo

Install binary:

    $ brew install servo-binary

Install from source:

    $ brew install --HEAD servo

For debug build:

    $ brew install --HEAD --with-debug servo

## To upgrade:

    $ brew update

Binary: 

    $ brew update servo-binary

Source:

    $ brew reinstall --HEAD servo
